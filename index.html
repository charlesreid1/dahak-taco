dahak-taco documentation¶

dahak-taco is an experimental command-line interface for running dahak workflows using Snakemake.

Getting Started¶

To get started with dahak-taco, and run your first workflow task, see Introduction: dahak-taco.

To figuure out what taco is doing and what it requires, see How Taco Works.
